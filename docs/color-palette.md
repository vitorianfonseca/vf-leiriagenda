# Paleta de Cores LeiriAgenda

## Cores Disponíveis

### Paleta Principal
- **Cream (#EDE0D4)** - Tom mais claro, ideal para fundos suaves
- **Sand (#E6CCB2)** - Tom claro, ótimo para cards e seções
- **Warm Tan (#DDB892)** - Tom médio-claro, para elementos secundários
- **Earthy Brown (#B08968)** - Cor primária, para botões e destaques principais
- **Rich Brown (#7F5539)** - Tom escuro, para estados hover e ênfase
- **Dark Brown (#9C6644)** - Tom mais escuro, para texto e bordas

## Como Usar no Tailwind

### Classes CSS Disponíveis
```css
/* Cores da paleta customizada */
bg-palette-cream
bg-palette-sand
bg-palette-warm-tan
bg-palette-earthy-brown
bg-palette-rich-brown
bg-palette-dark-brown

text-palette-cream
text-palette-sand
text-palette-warm-tan
text-palette-earthy-brown
text-palette-rich-brown
text-palette-dark-brown

border-palette-cream
border-palette-sand
border-palette-warm-tan
border-palette-earthy-brown
border-palette-rich-brown
border-palette-dark-brown
```

### Variáveis CSS
```css
--palette-cream
--palette-sand
--palette-warm-tan
--palette-earthy-brown
--palette-rich-brown
--palette-dark-brown
```

## Hierarquia de Uso Recomendada

1. **Cor Primária**: #B08968 (Earthy Brown) - Botões principais, links, logo
2. **Cor de Hover**: #7F5539 (Rich Brown) - Estados hover e ativo
3. **Fundos Suaves**: #EDE0D4 (Cream) - Seções, cards
4. **Elementos Secundários**: #DDB892 (Warm Tan) - Badges, elementos de destaque
5. **Texto e Bordas**: #9C6644 (Dark Brown) - Texto importante, bordas
